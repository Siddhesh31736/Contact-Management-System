# 📇 Contact Management System (Java Swing)

A desktop-based **Contact Management System** built using **Java Swing**.
This application allows users to **add, view, search, update, and delete contacts** with a simple and user-friendly interface.

---

## 🚀 Features

* ➕ Add new contacts
* 📋 View all contacts
* 🔍 Search contacts by name
* ✏️ Update existing contacts
* 🗑️ Delete contacts
* 💾 Data persistence using file storage (`contacts.txt`)
* 🖥️ GUI built with Java Swing

---

## 🛠️ Technologies Used

* **Java (JDK 8+)**
* **Java Swing (GUI)**
* **File Handling (BufferedReader, BufferedWriter)**
* **Eclipse IDE (recommended)**

## ▶️ How to Run

1. Clone the repository:

   ```
   git clone https://github.com/your-username/contact-manager-java.git
   ```

2. Open the project in Eclipse (or any Java IDE)

3. Navigate to:

   ```
   ContactGUI.java
   ```

4. Run the file as:

   ```
   Run As → Java Application
   ```
